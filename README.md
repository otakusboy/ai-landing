# ai-landing

Corporate landing page built with React, Vite, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project structure

- `src/components/layout/` — shared layout components
- `src/components/sections/` — page sections
- `src/components/ui/` — reusable UI primitives
- `src/data/` — content data arrays
